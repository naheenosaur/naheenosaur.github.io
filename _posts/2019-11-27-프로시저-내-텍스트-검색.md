## 텍스트검색

-   MySql
    ```sql
          -- 프로시저/함수 에서 텍스트검색
          SELECT * 
          FROM information_schema.routines 
          WHERE routine_definition LIKE '%키워드%' AND 
                routine_schema = 'DB명';
    ```
    
-   MSSQL
    ```sql
          -- 프로시저에서 텍스트검색
          SELECT * 
          FROM syscomments 
          WHERE text like '%키워드%';
    ```
    
    ```sql
          -- 테이블, 프로시저 등 생성날짜, 수정날짜 확인
          SELECT * 
          FROM sys.objects 
          ORDER BY modify_date DESC; 
    ```
